# AdminMod

A powerful and modern moderation mod for Minecraft Fabric, designed to help staff members manage players efficiently with an intuitive and customizable interface.

## ✨ Features

- 🛡️ Advanced moderation tools
- 🔨 Ban and mute management
- 📋 Punishment history tracking
- 👥 Alt-account detection
- 🎨 Fully customizable GUI
- ⚙️ Complete Mod Menu integration
- 📡 Discord webhook support
- 🚀 Lightweight and optimized performance
- 🔧 Easy configuration and setup

## 📸 Overview

AdminMod provides a clean and user-friendly moderation interface directly inside Minecraft, allowing staff members to manage punishments, review player history, and perform moderation actions quickly.

## 📦 Requirements

- Minecraft 1.21.11
- Fabric Loader
- Fabric API
- Mod Menu (Recommended)
- Java 21

## 🔧 Installation

1. Install Fabric Loader.
2. Download and install Fabric API.
3. (Optional) Install Mod Menu for easier configuration.
4. Download the latest version of AdminMod.
5. Place the `.jar` file into your `mods` folder.
6. Launch Minecraft.

## ⚙️ Configuration

AdminMod supports Mod Menu integration, allowing you to configure settings directly from Minecraft.

Available options include:

- GUI customization
- Moderation settings
- Discord webhook configuration
- Notification preferences
- Staff tool settings
